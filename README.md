# CodeAlpha_Social_Media_Platform
A feature-rich, full-stack social media platform built using Python and the Django web framework. This application allows the users to create accounts, share posts, interact with content, and connect with other users online.

## FEATURES:
-> User Authentication: Secure user registration, login, and logout functionality.
-> User Profiles: Personalized profile pages featuring user bios, avatars, and a collection of their individual posts.
-> Feed System: A dynamic timeline displaying the latest updates and posts from users across the platform.
-> Post Interaction: Ability to create, view, like, and comment on posts.
-> Responsive Design: Clean and accessible user interface optimized for both desktop and mobile viewing.

## TECH STACK:
-> Backend Framework: Django(Python).
-> Frontend Framework: HTML5, CSS3, JavaScript / Bootstrap.
-> Database: SQLite3 (Default development database).

## INSTALLATION AND SETUP:
Follow these steps to run the project locally on your machine;
1. CLONE THE REPOSITORY:
   ``` bash
   git clone https://github.com/A-Varsha29/CodeAlpha_Social_Platform.git
   cd CodeAlpha_Social_Platform

2. CREATE A VIRTUAL ENVIRONMENT:
   ``` bash
   python -m venv venv

3. ACTIVATE THE VIRTUAL ENVIRONMENT:
   ``` bash
   On Windows: venv\Scripts\activate
   On macOS/Linux: source venv/bin/activate

4. INSTALL DEPENDENCIES:
   Ensure you have Django installed in your environment:
   ``` bash
   pip install django

5. RUN DATABASE MIGRATIONS:
   ```bash
   python manage.py migrate

6. START THE DEVELOPMENT SERVER:   
     ``` bash
     python manage.py runserver

7. ACCESS THE APPLICATION:
      Open your browser and navigate to
   ```bash
   http://127.0.0.1:8000/
